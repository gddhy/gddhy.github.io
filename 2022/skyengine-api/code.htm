旧版本中的CODE范围定义

在老的版本中扩展接口code遵循着一下的范围约束。 

code 范围 说明
0-99 展讯平台
100-199 MTK平台
200-299 ADI平台
300-999 保留
1000-10000 多平台通用
新版本中CODE的组织方式

新版本中，将code划分成2个部分，高16位表示 group, 低16位表示功能。这样有效的将功能按组进行划分，原来的全部code对应的group为0。
