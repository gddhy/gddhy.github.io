uint32 mr_getTime(void)


Description
获取毫秒时间。
Parameters

Name Description

Return Value

当前的毫秒时间。

Remarks

获取系统时间，单位毫秒。系统时间可以以SkyEngine引擎启动之前的任意时刻为起始时间，返回从起始时间到目前经过的毫秒数。例如这个函数可以实现为返回系统启动后经过的毫秒数。

Example

/*mrp code*/

int32 tm = 0;
tm = mrc_getUptime();
mrc_printf( "time_ms = %d", tm );
