int32 mr_plat(int32 code = 1206, int32 param)


Description
获取手机当前设置的语言。
Parameters

Name Description
param 无

Return Value

MR_PLAT_VALUE_BASE + 0                   中文
MR_PLAT_VALUE_BASE + 1                   英文
MR_FAILED                                        获取失败
MR_IGNORE                                       不支持该功能
注：其他语言的键值可以放在“英文”之后返回。

Remarks

#define MR_PLAT_VALUE_BASE  1000

Example

/*mrp code*/

mrc_plat(1206, 0 );
