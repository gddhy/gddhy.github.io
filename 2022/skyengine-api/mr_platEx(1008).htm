int32 mr_platEx

(int32 code = 1008,
uint8* input,
int32 input_len,
uint8** output,
int32* output_len,
MR_PLAT_EX_CB* cb
)


Description
调用mtk的发送短信界面。
Parameters

Name Description
input T_PLAT_SENDSMS_REQ 数据结构指针
input_len T_PLAT_SENDSMS_REQ 数据结构大小
output 无
output_len 无
cb 无

Return Value

MR_SUCCESS                      成功
MR_FAILED                         失败
MR_IGNORE                        不支持该功能

Remarks

T_PLAT_SENDSMS_REQ数据结构：

typedef struct
{
    char *pNum;/*目前没有使用该参数*/
    char *pContent;/*大端unicode*/
}T_PLAT_SENDSMS_REQ;

Example

/*mrp code*/

T_PLAT_SENDSMS_REQ sendSMS = { 0 };
char wcontent[50];

str2wstr( (PCSTR)"hello", (PWSTR)&wcontent[0], (Sint32)sizeof(wcontent) );
sendSMS.pContent = wcontent;
mrc_platEx( 1008, (uint8*)&sendSMS, (int32)sizeof(T_PLAT_SENDSMS_REQ), NULL, NULL, NULL );
