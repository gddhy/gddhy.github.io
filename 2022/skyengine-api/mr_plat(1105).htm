int32 mr_plat(int32 code = 1105, int32 param)


Description
发送获得手机和sim卡存储空间的容量的请求。
Parameters

Name Description
param 无

Return Value

MR_WAITING                 发送请求成功
MR_FAILED                   调用失败

Remarks

处理结果通过mr_event返回给应用,mr_event的参数如下：
第一个：MR_SMS_OP_EVENT（15）；
第二个：MR_SMS_OP_GET_ME_SIZE ：表示第三个参数是手机上容量；
          MR_SMS_OP_GET_SIM_SIZE：表示第三个参数是sim卡上容量；
第三个：短信的容量。
enum
{
     MR_SMS_OP_GET_ME_SIZE,
     MR_SMS_OP_GET_SIM_SIZE,
     MR_SMS_OP_DELETE,
     MR_SMS_OP_GET_MSG_CONTENT
};

Example

/*mrp code*/

mrc_plat( 1105, 0 );

mr_event事件处理：
case MR_SMS_OP_EVENT:
{
switch ( wParam )
{
case MR_SMS_OP_GET_ME_SIZE :
  {
      //add your code here
  }
  break;
case MR_SMS_OP_GET_SIM_SIZE:
  {
      //add your code here
  }
  break;
default:
  break;
}
}
