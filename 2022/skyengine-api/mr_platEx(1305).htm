int32 mr_platEx

(int32 code = 1305,
uint8* input,
int32 input_len,
uint8** output,
int32* output_len,
MR_PLAT_EX_CB* cb
)


Description
获得指定盘符的剩余空间大小。
Parameters

Name Description
input 盘符标记（不区分大小写ASCII）
input_len 字符串长度
output T_DSM_FREE_SAPCE结构指针
output_len T_DSM_FREE_SAPCE结构长度
cb 无

Return Value

MR_SUCCESS            成功
MR_FAILED               失败
MR_IGNORE              不支持该功能
注：要得到以byte 为单位的大小值可以通过 account×unit 得到.

Remarks

Typedef struct
{
    uint32 total;    //总的大小
    uint32 tunit;    //总大小的单位
    uint32 account;  //剩余空间的大小
    uint32 unit;      //剩余大小的单位
}T_DSM_FREE_SAPCE

Unit：
1             ：单位是    byte
1024          ：单位是    k byte
1024x1024     ：单位是    m byte
1024x1024x1024：单位是    g byte

Example

/*mrp code*/

T_DSM_FREE_SAPCE *getInfo = NULL;
char *disk = NULL;
int32 len = 0;

 disk = "A";
 mrc_platEx( 1305, (uint8*)disk, sizeof(char), (uint8**)&getInfo, (int32*)&len, NULL );
 mrc_printf( "Info:totalSpace=%d/%d, freeSpace=%d/%d",
    getInfo->total, getInfo->tunit, getInfo->account, getInfo->unit );
