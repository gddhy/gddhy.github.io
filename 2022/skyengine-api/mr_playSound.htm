int32 mr_playSound(int type, const void* data, uint32 dataLen, int32 loop)


Description
播放音频。
Parameters

Name Description
type

音频类型，取值可以为：

   MR_SOUND_MIDI,

   MR_SOUND_WAV,

   MR_SOUND_MP3,

   MR_SOUND_AMR,

   MR_SOUND_PCM  //8K
data 音频数据
datalen 数据长度
loop !=0，音乐循环播放；==0，单次播放

Return Value

MR_SUCCESS               成功
MR_FAILED                  失败

Remarks

调用mr_playSound播放音频，调用mr_stopSound停止播放音频。

Example

/*mrp code*/

VOID* data;   // 音频数据流地址
int32 len;       //数据长度
mr_playSoundEx( MR_SOUND_MIDI, data, len, 0 );
