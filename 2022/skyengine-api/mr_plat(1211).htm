int32 mr_plat(int32 code = 1221, int32 param)


Description
调用平台接口产生随即数。
Parameters

Name Description
param 要产生的随即数的范围(正整数)

Return Value

MR_PLAT_VALUE_BASE + 随机数            产生随即数成功
MR_FAILED                                       产生随机数失败
MR_IGNORE                                      不支持该功能

Remarks

 

Example

/*mrp code*/

int32 randata = -1;

randata = mrc_plat( 1211, 100 );
mrc_printf( "Get random data: %d", randata );
