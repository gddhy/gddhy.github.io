int32 mr_getDatetime(mr_datetime* datetime)


Description
获取当前的日期。
Parameters

Name Description
datetime 日期时间

Return Value

MR_SUCCESS                       成功
MR_FAILED                          失败

Remarks

获取系统日期时间。
mr_datetime格式如下：
typedef struct
{
   uint16 year;                 //年
   uint8  month;                //月
   uint8  day;                  //日
   uint8  hour;                 //时，24小时制
   uint8  minute;               //分
   uint8  second;               //秒
}mr_datetime;

Example

/*mrp code*/

mr_datetime dt;
mrc_getDatetime( &dt );
mrc_printf( "Get date time is %d:%d:%d,%d:%d:%d", dt.year, dt.month, dt.day, dt.hour, dt.minute, dt.second );
