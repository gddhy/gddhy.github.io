int32 mr_platEx

(int32 code = 1316,
uint8* input,
int32 input_len,
uint8** output,
int32* output_len,
MR_PLAT_EX_CB* cb
)


Description
进行层叠加。
Parameters

Name Description
input T_LAYER_FLATTEN数据结构指针
input_len T_LAYER_FLATTEN数据结构大小
output 无
output_len 无
cb 无

Return Value

MR_SUCCESS                   成功
MR_FAILED                      失败
MR_IGNORE                     不支持该功能

Remarks

T_LAYER_FLATTEN数据结构：
typedef struct
{
    int32 desLayer;
    int32 layer1;
    int32 layer2;
    int32 layer3;
    int32 layer4;
}T_LAYER_FLATTEN;
注：将layer1，layer2，layer3，layer4，的数据叠加后放置在deslayer层中

Example

/*mrp code*/

T_LAYER_FLATTEN lflat = { 0 };
lflat.desLayer = baselayer;
lflat.layer1 = layer_hand1e_flatten1;
lflat.layer2 = layer_hand1e_flatten2;
mrc_platEx( 1316, (uint8*)&lflat, sizeof(T_LAYER_FLATTEN), NULL, NULL, NULL );
