int32 mr_seek(MR_FILE_HANDLE f, int32 pos, int method)


Description
设置文件读写指针。
Parameters

Name Description
f 文件句柄
pos 文件读写指针位置
method 设置方式,可能的取值为：
   MR_SEEK_SET, //从文件头开始搜索；
   MR_SEEK_CUR, //从文件尾开始搜索；
   MR_SEEK_END, //从文件当前位置开始搜索；

Return Value

MR_SUCCESS                    成功
MR_FAILED                       失败

Remarks

 

Example

/*mrp code*/

MR_FILE_HANDLE fh;
fh = mrc_open( "testFile.txt", MR_FILE_RDWR ); 
mrc_seek( fh, 10, MR_SEEK_SET);  //从文件头开始搜索；
mrc_seek( fh, 0, MR_SEEK_END);  //从文件尾开始搜索；
mrc_seek( fh, 10, MR_SEEK_CUR);  //从文件当前位置开始搜索；
mrc_close( fh );
