int32 mr_read(MR_FILE_HANDLE f, void *p, uint32 l)


Description
读取文件内容到指定的缓冲区。
Parameters

Name Description
f 文件句柄
p 文件缓存地址
l 缓冲长度

Return Value

>= 0                          读取的字节数
MR_FAILED                 失败

Remarks

初始化SkyEngine引擎，并运行指定的应用。当SkyEngine引擎启动时会调用mr_mem_get获取引擎需要的内存。 mr_mem_get函数由移植者实现，以取得引擎需要的内存。entry是SkyEngine入口参数。

Example

/*mrp code*/

MR_FILE_HANDLE fh;
char buffer[60];
fh = mrc_open( "testFile.txt", MR_FILE_RDWR );  //可读写方式打开文件testFile.txt
mrc_read( fh,  (void*)buffer, sizeof(buffer) );
mrc_close( fh );
