int32 mr_platEx

(int32 code = 1014,
uint8* input,
int32 input_len,
uint8** output,
int32* output_len,
MR_PLAT_EX_CB* cb
)


Description
申请扩展内存。
Parameters

Name Description
input 无
input_len 要申请的buffer的大小
output 申请到的buffer的指针
output_len 申请到的内存的大小（和input_len相等）
cb 无

Return Value

一直返回成功，需要判断*output的值。

Remarks

这个函数管理了2个屏幕大小的buffer，但是这个buffer在界面切换，或者被pause的时候需要主动释放掉。

Example

/*mrp code*/

int32 len;
int32 memor_len = 1024;
uint8 *pMemor = NULL;

mrc_platEx( 1014, NULL, memor_len, (uint8**)&pMemor, (int32*)&len, NULL );
mrc_printf(  "Applied memory interface:address = %d", pMemor );
