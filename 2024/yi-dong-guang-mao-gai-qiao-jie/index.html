移动改桥接

2024-03-12
字数统计:1.1k字 阅读时长≈3分

简单记一次移动光猫改桥接

光猫型号：H60G

这个型号的光猫开始没有动态管理员密码的，默认管理员账号 CMCCAdmin 密码 aDm8H%MdA ，在使用过程中发现移动后期改成了动态密码

可以先进入路由器后台查看，如果默认管理员密码可以使用，则可以跳过获取管理员密码步骤

获取管理员密码

先使用光猫背后的用户账号登录光猫管理员后台，依次打开 网络 - 远程管理 - 认证 ,记录下里面的 Password 的值，后续会用到

使用牙签或卡针捅光猫的Reset孔，多等一会松开，等光猫重置

光猫重置后使用电脑直连光猫lan口，打开光猫后台，此时可以使用光猫默认管理员账号登录

账号：CMCCAdmin
密码：aDm8H%MdA

登录依次打开 安全 - Telnet配置 - 勾选 启用LAN侧Telnet,记录下用户名、密码（也可自定义用户名密码），点击确定

右上角退出账号登录，选择 设备注册 ，输入刚才记录的password值，如果之前没有记录的话只能问宽带安装师傅询问注册码，输入注册码后等待移动下发数据注册网络

移动数据下发完成后光猫管理员密码会变成动态的，此时使用Telnet修改光猫管理员密码

按 Win + R 打开运行窗口输入 cmd 确定，打开控制台cmd窗口，输入下边命令后回车

出现telnet不是内部或外部命令的，去Windows可选功能里启用Telnet

输入前边记录的telnet用户名（回车）
输入前边记录的telnet密码（回车）

识符变成 ~$ 后输入： su （回车）

输入密码 aDm8H%MdA （回车）

标识符变成 /# 此时便是以root用户登录光猫控制台

修改管理员超级账号为 CMCCAdmin (可自定义)

修改管理员超级账号的密码 aDm8H%MdA (可自定义)

执行上面命令后管理员密码便改为了aDm8H%MdA或自定义的密码

光猫改桥接

使用管理员账号登录光猫后台，进入 网络 - 宽带设置 ，在链接名称里选择包含 _INTERNET_R_VID_ 的配置，记录下设备的 VLAN ID

记录VLAN ID后， _INTERNET_R_VID_ 配置可以保留也可以删除（建议直接删除）

在 连接名称 里选择 新建WAN连接

模式 选择 桥模式

VLAN模式 选择 改写(tag)

VLAN ID 里填写刚才记录的VLAN ID

使能打勾，端口绑定LAN1~4可以全打勾，也可以不选

DHCP服务器使能取消勾选

桥类型，我选择的 IP BRIDGE，不知道和PPPOE BRIDGE有何区别
业务模式选择INTERNET

设置完成后确定

改桥接后可以在路由器设置拨号，移动宽带账号一般为手机号，宽带密码可以询问宽带安装师傅或者自己重置密码

禁用移动的远程下发配置

在 网络 - 宽带设置 里，连接名称选择包含 TR069 的配置，进入浏览器调试（F12/Ctrl+shift+i）,找到使能、修改按钮，把HTML中的 disable 改为 enable 或者直接删掉。然后取消勾选使能，点击修改

在 网络 - 远程管理 - 省级数字家庭管理平台服务器 ,使用浏览器调试把使能和下边确定按钮改为可选择（方法同上），取消勾选使能，点击下边确定保存

修改完后可以再 状态 - 设备信息 里看到 ITMS注册状态 为 注册失败 便是成功

重置密码

编辑短信 KDMM 发送到 10086 查询

编辑短信 CZKDMM 发送到 10086 重置宽带密码

使用中国移动APP，打开 我的 - 我的家庭 - 家庭专区 -宽带服务 - 更多内容 - 密码修改

部分省市的默认密码 123123、123456、手机号后6位、身份证后6位

我参考恩山论坛帖子修改的，在这里简单记录，详细图文可以在下边帖子里看到
